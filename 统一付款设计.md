# 统一付款设计

> 平台统一收款、商户自行提现；用户可在购物车一次支付多店订单，每笔订单归属与金额记录清晰，便于商户对账与提现。

---

## 一、目标

- **用户**：购物车内多店商品可「统一付款」一次支付，无需按店多次去结算。
- **平台**：一笔支付对应多笔订单，钱进平台；每笔订单明确 storeId、金额、商品，商户端只看到自己的订单。
- **商户**：订单列表、收入、提现均按自己店铺的订单统计，记录无误。

---

## 二、数据与流程

### 2.1 统一支付记录（新集合 unified_payments）

| 字段 | 类型 | 说明 |
|------|------|------|
| _id | String | 自动生成 |
| out_trade_no | String | 微信支付商户订单号（唯一），用于统一下单与回调 |
| userOpenid | String | 下单用户 openid |
| orderIds | Array | 本笔支付包含的订单 _id 列表 |
| totalFeeFen | Number | 支付总金额（分） |
| status | String | pending / paid |
| createdAt | Date | 创建时间 |
| paidAt | Date | 支付成功时间（可选） |

- 一次「统一付款」生成一条 unified_payments，out_trade_no 用于调微信统一下单；支付成功后将该条置为 paid，并将 orderIds 中所有订单更新为 payStatus: 'paid'。

### 2.2 订单表（orders）扩展

- 增加可选字段：**unifiedPaymentId**（String，关联 unified_payments._id）。  
- 便于按「统一支付」维度查询、对账；商户侧仍按 storeId 查自己的订单即可。

### 2.3 流程简述

1. **购物车页**：用户点击「统一付款」→ 跳转「统一结算页」。
2. **统一结算页**：选地址、展示多店清单与总价 → 提交。
3. **云函数**：
   - **orderManage.createUnifiedOrders**：入参为 address + 多店 cart（每店 storeId、storeInfo、cartItems、cartTotal、deliveryFee、deliveryType）。为每店创建一笔订单（与现有 createOrder 逻辑一致，payStatus: 'unpaid'），并写一条 unified_payments（out_trade_no、orderIds、totalFeeFen、status: 'pending'），订单写入 unifiedPaymentId。返回 orderIds、totalFeeFen、unifiedPaymentId、out_trade_no。
   - **paymentManage.unifiedPrepay**：入参 unifiedPaymentId，校验 userOpenid 与 status，用 unified_payments.out_trade_no、totalFeeFen 调微信统一下单，返回小程序调起支付所需参数。
4. **前端**：wx.requestPayment；成功后调用 **orderManage.updateUnifiedPaymentPaid**（unifiedPaymentId），将对应 unified_payments 置为 paid，并将 orderIds 中全部订单更新为 payStatus: 'paid'；清空已支付店铺的购物车，跳转订单列表。

### 2.4 商户记录保证

- 每笔订单独立存在 orders 表，含 storeId、storeName、items、amountGoods、amountDelivery、platformFee、merchantIncome、payStatus 等。
- 商户端、统计、提现均按 storeId + 已支付订单查询，不依赖 unified_payments；unified_payments 仅用于「一次支付多单」的支付与对账。

---

## 三、涉及文件（实现时）

- 数据库：新增 unified_payments 集合；orders 增加 unifiedPaymentId（可选）。
- 云函数 orderManage：createUnifiedOrders、updateUnifiedPaymentPaid。
- 云函数 paymentManage：unifiedPrepay。
- 小程序：购物车页「统一付款」入口；统一结算页（地址、多店清单、总价、提交与调起支付、支付成功后更新并清空购物车）。
