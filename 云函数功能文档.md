## 用户相关

### 1. loginUser - 微信用户登录

**功能说明：** 处理微信小程序用户的登录和注册

**主要功能：**
- 微信登录凭证验证
- 自动创建新用户
- 更新用户最后登录时间
- 返回用户基本信息

**调用示例：**
```javascript
wx.cloud.callFunction({
  name: 'loginUser',
  data: {
    code: 'wx_login_code'
  }
});
```

**返回数据：**
- 用户ID
- 用户基本信息（昵称、头像等）
- 是否为新用户

## 订单相关

### 9. orderManage - 订单管理

**功能说明：** 处理所有订单相关的操作（功能最复杂的云函数）

**主要功能：**

#### 用户端功能：
- `createOrder` - 创建外卖订单
- `getOrderList` - 获取用户订单列表
- `cancelOrder` - 取消订单
- `getUserOrdersForReview` - 获取可评价订单

#### 商家端功能：
- `getMerchantOrders` - 获取商家订单列表
- `updateOrderStatus` - 更新订单状态（接单、制作中、待取餐等）
- `confirmMerchantReady` - 确认商家已准备好（待取餐状态）

#### 管理员功能：
- `getAdminOrderList` - 获取所有订单（管理端）

#### 跑腿/配送功能：
- `createExpressOrder` - 创建跑腿订单
- `createGamingOrder` - 创建代打游戏订单
- `createRewardOrder` - 创建悬赏订单
- `getReceiveOrders` - 获取可接单列表
- `getAvailableOrders` - 获取可抢单列表
- `grabOrder` - 抢单
- `getPickupOrders` - 获取待取货订单
- `confirmPickup` - 确认取货
- `getDeliverOrders` - 获取配送中订单
- `confirmDelivery` - 确认送达
- `completeOrder` - 完成订单

**调用示例：**
```javascript
// 创建订单
wx.cloud.callFunction({
  name: 'orderManage',
  data: {
    action: 'createOrder',
    data: {
      storeId: '店铺ID',
      cartItems: [...],
      address: {...},
      remark: '备注'
    }
  }
});

// 更新订单状态
wx.cloud.callFunction({
  name: 'orderManage',
  data: {
    action: 'updateOrderStatus',
    data: {
      orderId: '订单ID',
      status: 'preparing'
    }
  }
});
```

## 📝 返回格式说明

所有云函数都遵循统一的返回格式：

```javascript
{
  code: 200,        // 状态码：200成功，400参数错误，500服务器错误
  message: 'ok',     // 提示信息
  data: {}          // 返回数据
}
```

### 常见状态码：
- `200` - 操作成功
- `400` - 参数错误或业务逻辑错误
- `401` - 未授权（登录失败等）
- `403` - 权限不足
- `404` - 资源不存在
- `500` - 服务器内部错误

## ⚠️ 注意事项

1. **权限控制**：部分云函数需要管理员权限，调用前需验证身份
