## 🎨 页面设计特点

### 1. 管理端主页面 (admin-dashboard)

**功能特点：**
- 📊 **数据概览**：展示用户总数、商家数量、今日订单、今日营收等核心指标
- 🚀 **快捷操作**：提供商家管理、订单管理、用户管理、财务管理等快速入口
- 📈 **最近活动**：显示系统最新操作记录和事件
- 🔧 **系统状态**：监控数据库、支付系统等关键服务状态

**设计亮点：**
- 采用渐变背景和毛玻璃效果，营造现代感
- 卡片式布局，信息层次清晰
- 响应式网格系统，适配不同屏幕尺寸
- 丰富的图标和色彩搭配，提升用户体验

### 2. 商家管理页面 (admin-merchant-list)

**功能特点：**
- 🔍 **搜索筛选**：支持按商家名称、手机号搜索，按状态筛选
- 📋 **列表展示**：显示商家基本信息、状态、申请时间等
- ⚡ **快速操作**：一键审核、拒绝、暂停商家
- 📱 **响应式设计**：支持下拉刷新、上拉加载更多

**核心功能：**
- 商家状态管理（待审核、已通过、已暂停、已拒绝）
- 资质审核流程
- 商家详情查看
- 批量操作支持

### 3. 数据统计页面 (admin-statistics)

**功能特点：**
- 📊 **多维度统计**：数据概览、订单统计、用户统计、商家统计
- 📅 **时间筛选**：支持近一周、近一月、近三月数据查看
- 📈 **可视化展示**：图表化展示关键指标
- 📤 **数据导出**：支持统计数据导出功能

**统计维度：**
- 用户数据：总用户数、活跃用户、新增用户
- 订单数据：总订单数、完成率、取消率、营收统计
- 商家数据：总商家数、营业状态、审核状态

## 🔐 权限控制系统

### 权限验证机制
- **Token验证**：基于本地存储的管理员Token
- **过期检查**：自动检查登录状态和Token有效期
- **权限分级**：支持超级管理员和普通管理员角色
- **操作日志**：记录所有管理员操作行为

### 安全特性
- 密码加密存储（SHA256）
- 登录失败限制
- 操作审计日志
- 敏感信息保护

- **微信小程序原生开发**
- **WXML + WXSS + JavaScript**

- **响应式布局设计**

- **权限验证**：基于云函数的权限控制

```
用户操作 → 前端验证 → 云函数调用 → 数据库操作 → 结果返回 → 界面更新
```

## 🔧 开发指南

### 添加新功能页面
1. 在 `pages/` 目录下创建新页面文件夹
2. 实现 `.js`、`.wxml`、`.wxss`、`.json` 四个文件
3. 在 `app.json` 中注册页面路由
4. 在管理端主页面添加快捷操作入口

### 权限控制使用
```javascript
// 引入权限工具
const adminAuth = require('../../utils/adminAuth.js')

// 页面权限检查
onLoad() {
  adminAuth.checkAdminAuth().then(adminInfo => {
    // 权限验证通过，继续执行
  }).catch(error => {
    // 权限验证失败，跳转登录页
  })
}

// 特定权限检查
const canManageMerchants = await adminAuth.checkPermission('merchant_manage')
```

### 云函数调用
```javascript
// 调用统计云函数
const res = await wx.cloud.callFunction({
  name: 'statistics',
  data: { type: 'overview' }
})

// 调用商家管理云函数
const res = await wx.cloud.callFunction({
  name: 'merchantManage',
  data: {
    action: 'getList',
    data: { page: 1, pageSize: 10 }
  }
})
```
