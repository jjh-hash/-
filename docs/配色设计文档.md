# 校园外卖 - 配色设计文档

基于 [kigen.design/color](https://kigen.design/color) 生成的色板整理，统一主色、副色、次色与强调色，便于全项目一致使用。

**当前方案：方案 2 — 橙/暖黄为主色**，更符合大学生审美，突出活力、食欲与年轻感。

---

## 一、设计原则

| 角色 | 用途 | 使用场景 |
|------|------|----------|
| **主色 (Primary)** | 品牌、主要操作、选中态 | 导航栏、Tab 选中、主按钮、链接、结算栏「结算」按钮 |
| **副色 (Secondary)** | 次要操作、信息强调、分类点缀 | 代拿快递卡片、联系商家、部分图标与标签 |
| **次色 (Neutral)** | 背景、边框、正文与辅助文字 | 页面背景、卡片底、分割线、正文/副文/占位符 |
| **强调色 (Accent)** | 状态与反馈 | 成功(绿)、警告/待支付(深橙)、危险/取消(红) |

- 主色控制「品牌感」，其余不抢主色。
- 同一界面主色面积 > 副色 > 强调色；次色用于承载内容与层次。

## 五、强调色 - 状态与反馈

### 5.1 成功 (Success)

主色已为橙/黄，成功态用**绿色**区分，表示「完成、通过」。

- 主值：`#07C160`（Emerald 绿）
- 浅底：`#C2FFD1` 或 `rgba(7, 193, 96, 0.12)`

### 5.2 警告 / 待支付 (Warning)

与主色同系但更深，用于「待支付、倒计时、即将超时」等需注意但非危险的状态。

- 主值：`#DD8E09` 或 `#BE7906`（主色橙 500/600，深橙/琥珀）
- 浅底：`#FFEEE2`（主色 50）

### 5.3 危险 / 取消 (Danger)

**选用色板：Cinnabar**（基础色 `#EF4444`。）

| 色阶 | Hex | 用途建议 |
|------|-----|----------|
| 50 | `#FCEDED` | 危险浅底、错误提示背景 |
| 100 | `#F9DADA` | 危险 hover |
| 200 | `#F2B5B5` | — |
| 300 | `#EF4444` | **危险主值**：取消、删除、错误、角标 |
| 400 | `#E03D3D` | 危险 hover |
| 500 | `#C73535` | — |
| 600–950 | （略） | 深色备用 |

## 六、与现有 CSS 变量对照

当前 `app.wxss` 中已使用的变量建议与色板对应关系如下，便于后续替换为上述 hex。（主色已切换为橙/暖黄方案。）

| 变量名 | 建议取值 | 说明 |
|--------|----------|------|
| `--color-primary` | `#FFB800` | 主色（亮黄，推荐）/ 或 `#F59E0B`（橙） |
| `--color-primary-light` | `#FFEEE2` 或 `rgba(255, 184, 0, 0.12)` | 主色浅底（Orange Peel 50） |
| `--color-success` | `#07C160` | 成功态保持绿（Emerald） |
| `--color-warning` | `#DD8E09` 或 `#BE7906` | 警告/待支付（深橙，与主色区分） |
| `--color-danger` | `#EF4444` | Cinnabar 300 |
| `--color-text` | `#31353C` 或 `#1B1D22` | Pale Sky 800 / 900 |
| `--color-text-secondary` | `#6B7280` | Pale Sky 500 |
| `--color-text-placeholder` | `#9AA2B1` | Pale Sky 300 |
| `--color-bg-page` | `#FFF0E0` | 暖黄略深（蜜桃/杏色底，与主色橙黄搭） |
| `--color-bg-card` | `#FFFFFF` | 保持白 |
| `--color-border` | `#B8BDC8` 或 `#E5E7EB` | Pale Sky 200 或略浅 |

### 新增变量建议（副色与扩展）

若需在 WXSS 中直接使用副色与更多色阶，可增加：

```css
/* 副色 - 蓝 */
--color-secondary: #3B82F6;
--color-secondary-light: #EDF0FE;

/* 主色扩展 - 橙/暖黄（可选） */
--color-primary-50: #FFEEE2;
--color-primary-100: #FFD9BE;
--color-primary-400: #FFB800;
--color-primary-500: #DD8E09;

/* 中性灰扩展（可选） */
--color-neutral-50: #F0F1F3;
--color-neutral-200: #B8BDC8;
--color-neutral-500: #6B7280;
--color-neutral-800: #31353C;
```

## 八、渐变配色（参考）

以下为搜集的「舒服的渐变」方案，均为三色渐变，可直接用于 Banner、空状态、卡片背景、弹窗头部等，提升氛围感。主色为橙/暖黄，可优先选与主色或副色蓝相近的几组。

| 名称 | 色值（左 → 右） | 风格 | 建议场景 |
|------|-----------------|------|----------|
| **粉桃** | `#FF5391` → `#FFB09C` → `#FF9CDA` | 暖粉、柔和 | 活动 Banner、女性向/节日氛围 |
| **紫橙黄** | `#B18DFF` → `#FFBB57` → `#FFF443` | 活泼、高对比 | 促销/闪购、游戏陪玩入口 |
| **粉蓝紫** | `#FD91FF` → `#5240FE` → `#AD9EFF` | 梦幻、科技感 | 空状态插图底、弹窗装饰 |
| **金与蓝** | `#FFC870` → `#87BDFF` → `#DBEAFF` | 温暖到清凉 | 首页顶部氛围、订单完成页 |
| **天蓝薰衣草** | `#58B8FF` → `#74DFFF` → `#90BBFF` | 清爽、信任感 | 代拿快递/跑腿模块、通用 Banner |
| **青柠薄荷黄** | `#CEF595` → `#B1FFEC` → `#FFE588` | 清新、自然 | 与主色绿最接近，推荐首页/分类卡片背景 |

### CSS 使用示例

```css
/* 线性渐变（水平） */
.gradient-pink { background: linear-gradient(90deg, #FF5391 0%, #FFB09C 50%, #FF9CDA 100%); }
.gradient-purple-yellow { background: linear-gradient(90deg, #B18DFF 0%, #FFBB57 50%, #FFF443 100%); }
.gradient-mint { background: linear-gradient(90deg, #CEF595 0%, #B1FFEC 50%, #FFE588 100%); }

/* 线性渐变（对角，更柔和） */
.gradient-gold-blue { background: linear-gradient(135deg, #FFC870 0%, #87BDFF 50%, #DBEAFF 100%); }
.gradient-sky { background: linear-gradient(135deg, #58B8FF 0%, #74DFFF 50%, #90BBFF 100%); }
```

**建议**：主色为橙/暖黄，优先使用 **金与蓝**、**紫橙黄** 与主色呼应；**青柠薄荷黄** 可作辅助氛围；代拿/跑腿模块可用 **天蓝薰衣草**；活动/促销可用 **粉桃** 增强吸引力。

## 九、色板来源说明

- 所有色阶均由 **kigen.design/color** 生成。
- 算法：**Tailwind CSS**；命名：**50,100…900,950**；色阶数：**11**。
- **主色**：Orange Peel（#F59E0B），主值推荐 **#FFB800** 亮黄，更符合大学生审美。
- **副色**：Azure Radiance（#3B82F6）；**次色**：Pale Sky（#6B7280）；**成功**：Emerald（#07C160）；**警告**：主色深阶（#DD8E09/#BE7906）；**危险**：Cinnabar（#EF4444）。
- 渐变配色为设计参考搜集，可自由选用并微调角度与色阶位置。

后续若在 kigen 中微调 Seed Color 或 Contrast Shift，只需替换本文档中对应 hex，并同步到 `app.wxss` 中的变量即可。
